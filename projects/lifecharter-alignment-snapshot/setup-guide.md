### Step 4: Configure Rating Scale for All Questions

**For EACH question, set up this 1-5 scale:**

| Score | Label |
|-------|-------|
| 1 | Deeply misaligned - This statement does not feel true in this season |
| 2 | Strained - This statement is rarely true or feels difficult to access |
| 3 | Unclear or inconsistent - This statement is sometimes true, but not steady |
| 4 | Mostly aligned - This statement is generally true and supported |
| 5 | Deeply aligned - This statement feels true, embodied, and alive |

**How to Configure:**
1. Open each question
2. Look for **"Scale Options"** or **"Rating Labels"**
3. Enter the labels above for scores 1-5
4. Save each question

### Step 7: Configure Scoring (If Needed)

**Scoring Formula:**
- Each answer: 1-5 points
- Total possible: 60-300 points
- Dimension Score: (Sum of 5 answers / 5) / 5 × 100

**QuizForma should calculate this automatically** with rating scale questions.

**Verify scoring is enabled:**
1. Check **"Scoring"** or **"Calculations"** section
2. Ensure **"Calculate Score"** is enabled
3. Verify total score range is 60-300

## 🆘 TROUBLESHOOTING

### QuizForma Issues:

**Questions not saving:**
- Check internet connection
- Clear browser cache
- Try different browser

**Scoring not working:**
- Verify all questions are rating scale type
- Check that point values are set (1-5)
- Ensure result categories have correct score ranges

**Lead capture not showing:**
- Enable lead capture in settings
- Check that form fields are configured
- Verify redirect is set to result categories

### Page Sprout Issues:

**Page not publishing:**
- Check all required fields are filled
- Verify URL slug is unique
- Save all changes before publishing

**Design not applying:**
- Clear browser cache
- Check that colors are valid hex codes
- Preview page to verify changes

**Buttons not working:**
- Verify URLs are correct (include https://)
- Test links in incognito mode
- Check for typos in URLs
