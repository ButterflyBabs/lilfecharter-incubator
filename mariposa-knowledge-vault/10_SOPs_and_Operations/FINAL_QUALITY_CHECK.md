# Mariposa Final Quality Check

**Pre-Delivery Review Checklist**

*Created: May 9, 2026*
*Last Updated: May 9, 2026*

---

## Purpose

Before returning any final work to Babs, review it through this 13-point checklist. This ensures quality, accuracy, and alignment with Sacred Kaleidoscope Community standards.

---

## The 13-Point Quality Checklist

### 1. Task Completion
**Is the task fully answered?**
- [ ] All requirements addressed
- [ ] No missing components
- [ ] Deliverable matches request

### 2. Business Area Alignment
**Is the correct business area identified?**
- [ ] Sacred Kaleidoscope Community
- [ ] LifeCharter
- [ ] Conversations of Consequence
- [ ] Life by Design Summit
- [ ] Embraced:Fully
- [ ] Reentry/Angel Teams
- [ ] General Operations

### 3. Primary Agent Assignment
**Is the correct Primary Agent assigned?**
- [ ] Agent expertise matches task
- [ ] Agent has necessary skills
- [ ] Agent capacity considered

### 4. Support Agent Necessity
**Are the Support Agents necessary and limited?**
- [ ] Maximum 3 support agents
- [ ] Each support agent has clear role
- [ ] No unnecessary agents assigned

### 5. Deliverable Clarity
**Is the final deliverable clear?**
- [ ] Specific format defined
- [ ] Scope boundaries clear
- [ ] Success criteria stated

### 6. Voice/Tone Alignment
**Is the voice/tone aligned with the correct brand?**
- [ ] Babs' voice (compassionate, metaphysical, practical, warm)
- [ ] Brand guidelines followed
- [ ] Tone appropriate for audience

### 7. Practical Usability
**Is the work practical and usable?**
- [ ] Ready for immediate use
- [ ] No additional work required
- [ ] Clear implementation steps

### 8. Factual Accuracy
**Are names, links, dates, prices, event times, and program titles accurate?**
- [ ] All names spelled correctly
- [ ] All links tested and working
- [ ] Dates and times verified
- [ ] Prices current and correct
- [ ] Program titles exact

### 9. Call-to-Action Clarity
**Is the CTA clear, if a CTA is needed?**
- [ ] Single, clear action
- [ ] Link provided and tested
- [ ] Urgency appropriate (not pushy)

### 10. Approval Requirements
**Does anything require Babs' approval before use?**
- [ ] Checked against Approval Matrix
- [ ] Approval flagged if needed
- [ ] Safe next step provided

### 11. Sensitivity Review
**Is there any sensitive content involving:**
- [ ] Trauma
- [ ] Disability
- [ ] Incarceration
- [ ] Reentry
- [ ] Family
- [ ] Finances
- [ ] Legal matters
- [ ] Medical matters
- [ ] Public reputation

*If YES, flag for Babs review*

### 12. Risk Assessment
**Are there any promises, risks, assumptions, or missing details that need to be flagged?**
- [ ] No overpromising outcomes
- [ ] No assumptions unstated
- [ ] No missing critical details
- [ ] Risks identified and flagged

### 13. Next Step Clarity
**Is the next step obvious?**
- [ ] Clear action item
- [ ] Responsible party identified
- [ ] Timeline specified

---

## Quality Check Response Format

After completing the checklist, respond with:

```
QUALITY CHECK COMPLETE:
✅ [Number] of 13 items passed

FLAGGED ITEMS:
[List any items that need attention]

READY FOR DELIVERY:
Yes / No - with explanation

RECOMMENDED ACTION:
[What should happen next]
```

---

## The Four Do Nots

**Do not bury warnings.**
- Flag concerns prominently, not in footnotes

**Do not pretend certainty.**
- If unsure, say "I recommend verifying X"

**Do not over-explain.**
- Be clear and concise

**Do not hand Babs a messy pile.**
- Return clean, organized, ready-to-use work

---

## Delivery Standards

Return work that is:
- ✅ Clean
- ✅ Useful
- ✅ Aligned with brand and mission
- ✅ Ready for immediate use or clear next step

*This checklist protects quality, maintains trust, and ensures every deliverable meets Sacred Kaleidoscope Community standards.*
