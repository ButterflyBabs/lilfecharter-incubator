# Content Routing Refinement

**Specialized Routing Rules for Specific Content Types**

*Created: May 9, 2026*
*Last Updated: May 9, 2026*

---

## Conversations of Consequence DAILY Episode Packages

### Primary Agent
**Consequence Curator**

### Default Support Agents
1. **Story Steward** - for message arc, teaching flow, story depth, and long-form content structure
2. **Babs Voicekeeper** - for Babs voice, spiritual grounding, tenderness, clarity, and trauma-informed tone

### Conditional Support Agent
3. **Curiosity Catalyst** - ONLY if the episode connects to a campaign, offer, event, or conversion goal

### When to Use Sumbal
Use Sumbal **only** when the task involves:
- Uploading or formatting inside Spotify, YouTube, GHL, or another platform
- Setting up automations
- Checking links
- Testing publication flow
- Troubleshooting tech
- Creating implementation instructions for a human tech assistant

**Do NOT use Sumbal as a default support agent for writing or content strategy.**

---

## Routing Decision Tree

```
Is this a Conversations of Consequence DAILY episode?
├── YES
│   ├── Primary: Consequence Curator
│   ├── Support 1: Story Steward (always)
│   ├── Support 2: Babs Voicekeeper (always)
│   └── Support 3: Curiosity Catalyst (only if campaign/offer/event connected)
│
│   Does this involve platform upload/formatting/tech?
│   ├── YES → Add Sumbal
│   └── NO → Do not add Sumbal
│
└── NO → Use standard routing rules
```

### Task: "Create a daily episode about the Yellow Light"

**Support 3:** None (no campaign/offer connection)

### Task: "Create a daily episode about the Yellow Light AND promote the upcoming summit"

**Primary:** Consequence Curator

**Support:**
1. Story Steward (message arc and teaching flow)
2. Babs Voicekeeper (voice and spiritual grounding)
3. Curiosity Catalyst (summit promotion angle)

**Sumbal:** No (writing phase, not tech implementation)

---

### Task: "Upload today's episode to Spotify and YouTube"

**Primary:** Sumbal

**Support:**
1. Consequence Curator (verify content accuracy)
2. Babs Voicekeeper (final voice check if needed)

---

## Key Principle

**Match the agent to the actual work being done.**

- Writing → Content agents (Consequence Curator, Story Steward, Babs Voicekeeper)
- Strategy → Strategy agents (Curiosity Catalyst)
- Tech implementation → Tech agents (Sumbal)
- Admin → Admin agents (KeeJay)

Do not default to tech agents for writing work. Do not default to content agents for tech work.

---

*This refinement ensures the right expertise is applied to the right phase of work.*
