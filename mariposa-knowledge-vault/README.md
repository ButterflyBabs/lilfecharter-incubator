## Usage Rules

1. **Consult First**: Before starting any task, check the relevant Knowledge Vault section
2. **Maintain Consistency**: Use the Vault to ensure voice, language, and formatting consistency
3. **Flag Gaps**: If information is missing, flag it clearly and recommend additions
4. **Update Regularly**: The Vault should evolve as the business grows
5. **Source of Truth**: When in doubt, the Vault overrides general knowledge

---

## Missing Knowledge Protocol

If a task requires information not in the Knowledge Vault:

1. **Flag clearly**: "This task requires information not currently in the Knowledge Vault"
2. **Specify section**: "Section [XX] needs [specific information]"
3. **Recommend addition**: "I recommend adding [details] to maintain consistency"
4. **Proceed with caution**: Use best judgment, but note the gap for future filling
