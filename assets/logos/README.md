# LifeCharter Brand Assets

## Official Logo

**File:** `lifecharter-logo-official.jpg`

**Usage:** Primary logo for LifeCharter Alignment Snapshot and all LifeCharter-related web properties.

**Background:** The logo works best with a Paper-colored background (#F5F1E8) to match the Sacred Kaleidoscope brand palette.

**Location in Projects:**
- Copy to: `public/lifecharter-logo.jpg`
- Reference in HTML: `<img src="lifecharter-logo.jpg" alt="LifeCharter" class="logo-img">`

**CSS for Logo Display:**
```css
.logo-container {
    margin-bottom: 15px;
    background-color: var(--paper);
    padding: 15px;
    border-radius: 8px;
    display: inline-block;
}

.logo-img {
    max-width: 280px;
    height: auto;
    display: block;
    margin: 0 auto;
    background-color: var(--paper);
}
```
